## 🆕 本轮（05:00 后）新增 / 修改

### 后端
- **持久化扩展**：`InMemoryDataRepository` 与 `PersistenceStore` 新增面试分类、面试记录两个领域，全量与 SQLite 同步落盘（启动装载 → 退出/定时落库），重启数据不丢。
- **管理员可配置项**：`SystemConfig` 新增 `interviewTotalMinutes / interviewMaxQuestions / interviewDailyLimit / interviewOpening / interviewSelfIntroPrompt / interviewSystemPrompt` 六个字段；默认值合理、向后兼容。
- **`InterviewService` 重写**：
  - `generateNextQuestion`：把简历摘要 + 历史问答 + 分类提问方向拼成 prompt 交给 `AiHttpClient`；AI 出错/为空时本地兜底文案。
  - `submitAndEvaluate`：把所有问答打包成 JSON Schema 严格输出的 prompt，让 AI 一次性返回逐题分、薄弱点、建议、综合分、能力分布、总结、鼓励语；AI 失败则按答题长度本地评分兜底。落库到 `rl_interview_record`。
  - `getRemainingQuota`：基于 SystemConfig 的每日上限 + 仓库当日使用次数计算。
- **REST 控制器**：
  - `/interview/categories` `/interview/config` `/interview/question` `/interview/submit` `/interview/records*` `/interview/quota`
  - `/admin/interview-categories`（GET/POST/PUT/DELETE）

### 前端
- **`AdminView` 缩减一个 Tab**：把原有 `模板分类` 合并进 `模板管理`（子分段切换）；新增 `模拟测试` Tab，承载面试分类 CRUD + 全局配置（总时长、最大题数、每日上限、开场白、自我介绍 prompt、AI 人设 prompt）。
- **`InterviewView` 全面重写**：沉浸式聊天界面（豆包 / ChatGPT 风格），面试官头像换成西装人像；总时长由后端配置驱动；单题不限时；倒计时器 + 气泡阴影 + 思考动画；结束后展示综合分、能力雷达图（ECharts）、逐题详情、鼓励语、报告下载。
- **`ProfileView` 新增「面试历史」Tab**：列表卡片 + 详情弹窗（Hero 综合分卡 + 能力雷达 + 逐题展开 + 建议），支持删除；按 `?tab=interview` 直达。
- **`ShareView` 重做样式**：加 glass header + 大圆角卡片 + 浏览徽标 + 渐变底纹。

### 联调
- 后端 Maven 编译通过，前端 Vite build 通过。
- 启动 8080 后端 + 5174 前端，curl 跑通：
  - 公开：`categories / config / quota / question / submit / records / records/{id} / DELETE records/{id}` 全部 200。
  - 后台：`auth/login → interview-categories CRUD → system-config PUT → 公开 config GET` 链路全部 200。
